## Methods

| Methods               |        Description                       |
| ----------------- | ---------------------------------------------------------------- |
| GET               | Returns information from one or more records. |
| POST              | Used to create a new record. |
| PUT               | Update data in a record or change its status. |
| DELETE            |Removes a record from the system.|
